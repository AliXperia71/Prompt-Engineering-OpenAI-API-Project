# Prompt-Engineering-OpenAI-API-Project

This project focuses on generating an OpenAI API key and building a custom Flashcard prompt tester app. I first start by pip installing OpenAI and exporting my custom API key, then I import OpenAI into my project, setting it into my client variable, then test sample prompts e.g. some information on the popular video game: Minecraft, to generate 3 different types of flashcards based on the prompt instructions. 
Finally after the program is executed, all the work is saved in a .json custom file that is saved as experimental_log_[date done]_[time done].json format.
